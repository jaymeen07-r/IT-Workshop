# College Information Website

## Overview
This project is a simple **multi-page HTML website** for a college, showcasing basic information about departments, courses, faculties, and library facilities.  
It is built using **HTML** only, without any external frameworks or libraries.

---

## Pages Included

| Page | Description |
|------|-------------|
| `index.html` | Home page with navigation links to all sections |
| `departments.html` | List of all departments in the college |
| `courses.html` | Undergraduate engineering courses offered with intake details |
| `faculties.html` | Principal, HODs, and selected faculty members |
| `library.html` | Library facilities including digital and physical resources |

## Features
- Simple **navigation menu** linking all pages
- Clean **HTML structure** for easy readability and modification
- Provides **information about courses, departments, faculties, and library** in a structured format
- Each page has a **link back to Home** for easy navigation

# Resume Website

## Overview
This project is a simple **single-page Resume Website** created using **HTML and CSS**.  
It presents personal details, skills, education, projects, internships, and achievements in a clean and professional layout.

The website is lightweight, responsive, and suitable for beginners learning front-end development.

---

## Files Included

| File | Description |
|------|-------------|
| `index.html` | Main resume page containing all personal and professional information |
| `style.css` | CSS file for styling, layout, and visual enhancement |
| `README.md` | Project documentation |
